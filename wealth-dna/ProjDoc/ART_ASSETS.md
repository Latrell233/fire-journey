# Oracle (财务基因解码) — 美术素材资产需求清单

> 优先级: P0 = 核心体验必需, P1 = 显著提升品质, P2 = 锦上添花

---

## 🔴 P0 — 核心体验缺失

### 1. Favicon / App Icon
| 项目 | 规格 |
|------|------|
| **用途** | 浏览器标签页图标、添加到主屏幕图标 |
| **格式** | SVG (源文件) + PNG (导出) |
| **尺寸** | 16×16, 32×32, 180×180 (apple-touch-icon), 192×192, 512×512 |
| **风格** | 简约几何图形 — 建议用 DNA 双螺旋 + 金币/火焰元素抽象组合 |
| **配色** | 主色 `#d4a04a`（金），辅助 `#2c2c2c`（深棕） |
| **参考** | Notion、Duolingo 的极简几何风 app icon |

### 2. OG 分享图
| 项目 | 规格 |
|------|------|
| **用途** | 微信/社交媒体分享时展示的卡片图 |
| **格式** | PNG |
| **尺寸** | 1200×630 (标准 OG) + 300×300 (微信) |
| **内容** | 「财务基因解码 · Wealth DNA」标题 + 简短 sloga + 品牌色背景 |
| **风格** | 干净、高级感，避免廉价测试类广告感 |

## 🟡 P1 — 显著提升品质

### 3. 流派横幅插图 (Faction Banner) ×4
| 项目 | 规格 |
|------|------|
| **用途** | 结果页顶部彩色横幅的背景图 |
| **当前降级** | 纯色块 (`#d4a04a` 等) |
| **格式** | PNG (带透明通道) 或 SVG |
| **尺寸** | 750×480 (2x) — 移动端全宽横幅 |
| **风格** | 每张图传达对应流派的气质： |

| 流派 | 文件名 | 色彩 | 意象关键词 |
|------|--------|------|-----------|
| FE 拓荒者 | `fe.png` | 暖金 `#d4a04a` | 风暴、帆船、荒野、探索、自由 |
| FO 离岸者 | `fo.png` | 雾蓝 `#7a9ca8` | 海岸、灯塔、独木舟、宁静、退隐 |
| VE 征服者 | `ve.png` | 赤褐 `#8b5a4a` | 王座、山巅、城堡、竞争、荣耀 |
| VO 守夜人 | `vo.png` | 苔绿 `#6b7a5c` | 城墙、盾牌、家园、炉火、守护 |

**技术要求**:
- 图片上部留出足够的暗区/虚化区，确保白色文字可读
- 文件大小 ≤ 80KB（WebP 备选）
- 建议用半抽象几何风格，避免过于写实

### 4. 人物插画 (Character Illustration) ×16
| 项目 | 规格 |
|------|------|
| **用途** | 结果页角色形象展示 |
| **当前降级** | `onError` 隐藏，完全不显示 |
| **格式** | PNG (透明背景) 或 SVG |
| **尺寸** | 400×400 (2x) — 高度约束 200px 展示 |
| **风格建议** | 半身像 / 头像徽章风格 |

16 个人格类型：

```
FE 拓荒者:                   FO 离岸者:
  S-I-F-E  远航-拓荒者         S-I-F-O  蓄海-离岸者
  S-G-F-E  筑城-拓荒者         S-G-F-O  锚泊-离岸者
  C-I-F-E  烈焰-拓荒者         C-I-F-O  巡游-离岸者
  C-G-F-E  逍遥-拓荒者         C-G-F-O  栖枝-离岸者

VE 征服者:                   VO 守夜人:
  S-I-V-E  铸碑-征服者         S-I-V-O  铸垒-守夜人
  S-G-V-E  磐石-征服者         S-G-V-O  沉锚-守夜人
  C-I-V-E  烈焰-征服者         C-I-V-O  破局-守夜人
  C-G-V-E  聚光-征服者         C-G-V-O  织光-守夜人
```

**风格方向**（择一）:
- **A. 极简几何人像** — 纯色块 + 几何形状拼出人物气质（Picasso 感），AI 可批量生成
- **B. 线条插画** — 单色线条勾勒，类似 Notion/Linear 的插画风格，配合流派色
- **C. 微写实半身** — 简约面部 + 代表性服装/配饰，用流派色做背景光晕

> 建议先用方案 A 或 B 快速产出 16 张，后续迭代升级为方案 C。

## 🟢 P2 — 锦上添花

### 5. 维度图标 ×4
| 项目 | 规格 |
|------|------|
| **用途** | 四维解读卡片上的小图标 |
| **格式** | SVG (内联) |
| **尺寸** | 24×24 / 32×32 |
| **配色** | 单色 `#d4a04a`（主色） |

| 维度 | 代号 | 图标方向 |
|------|------|---------|
| SC 资源分配观 | 蓄水 vs 体验 | 💧 水滴 / 💸 消费 |
| IG 风险耐受力 | 进取 vs 防守 | 📈 上升箭头 / 🛡️ 盾牌 |
| FV 核心内驱力 | 自由 vs 认同 | 🕊️ 翅膀 / 🏆 奖杯 |
| EO 终极价值观 | 激荡 vs 秩序 | ⚡ 闪电 / 🏠 家园 |

### 6. DNA 解码动效素材
| 项目 | 规格 |
|------|------|
| **用途** | `DnaReveal` 组件 — 答题完成后的"计算中"过渡动画 |
| **当前降级** | CSS 圆环旋转 |
| **格式** | Lottie JSON 或帧序列 PNG/WebP |
| **时长** | 2-3 秒循环 |
| **内容** | 双螺旋旋转/粒子汇聚/基因序列扫描效果 |
| **尺寸** | 200×200 (视口内展示约 100×100) |

### 7. 背景纹理
| 项目 | 规格 |
|------|------|
| **用途** | 页面背景微纹理，替代纯 `#surface` 色 |
| **格式** | PNG 或 SVG pattern |
| **尺寸** | 可平铺 (200×200 或纯 CSS) |
| **风格** | 极淡的噪点/纸纹/网格纹理，`opacity: 0.03` 级别 |
| **参考** | Linear.app 的噪点背景、Notion 的纸纹理 |

### 8. 分享卡片装饰
| 项目 | 规格 |
|------|------|
| **用途** | `ShareCard` 组件的背景装饰元素 |
| **格式** | SVG |
| **尺寸** | 小型装饰性几何图形（星、环、点阵） |
| **风格** | 配合流派色，简洁几何风 |

## 📐 全局设计约束

| 约束 | 值 |
|------|-----|
| **设计宽度** | 375px (移动端基准)，最大 480px |
| **品牌主色** | `#d4a04a` (暖金) |
| **品牌辅助色** | `#2c2c2c` (深棕黑), `#6b6258` (中灰棕), `#f0ebe4` (暖白) |
| **流派色** | FE 金 `#d4a04a` / FO 蓝 `#7a9ca8` / VE 褐 `#8b5a4a` / VO 绿 `#6b7a5c` |
| **圆角** | 卡片 `12px`、按钮 `8px` |
| **字体** | 系统字体栈 (苹方/PingFang SC 优先) |
| **动效时长** | 快速 150-180ms、标准 300ms、仪式感 600-800ms |
