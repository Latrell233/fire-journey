品牌总述 (Brand Overview)
中文品牌名：财富自由之旅
英文品牌名：The FIRE Journey
核心理念：FIRE (Financial Independence, Retire Early)。我们不仅仅提供一个工具，而是为用户提供一套从“财务认知觉醒”到“人生沙盘推演”，最终落地为“真实资产掌控”的完整闭环系统。
品牌调性：数据驱动、沉浸式、强交互反馈。

产品矩阵与模块定义 (Product Matrix)
The FIRE Journey 由三个紧密耦合但独立运行的子产品组成，它们共同构成了用户通往财务自由的完整旅程：
模块代号   产品名称   定位与形态   核心价值
Oracle   财务基因解码 (Wealth DNA)   引流层 / H5轻应用48题沉浸式人格测试   通过两难抉择挖掘用户潜意识，输出专属“4位DNA代码”与人格报告，实现低门槛裂变传播。

Sandbox   人生推演沙盘 (Life Sandbox)   体验层 / 互动游戏回合制文字冒险模拟器   承接DNA数据，以游戏化方式推演用户未来30年的资产曲线。强调决策的重量感与延迟反馈。

Compass   资产罗盘 (Asset Compass)   留存层 / 效率工具真实资产管理App   将沙盘中的推演转化为现实行动。提供真实的资产追踪、复盘与路线校准功能。

关联性设计 (Interconnectivity)

** 产品逻辑关联 (User Journey)**
三大模块遵循严密的漏斗模型，环环相扣：
觉醒 (Oracle)：用户被“测测你的财务DNA”吸引进入，完成测试并获得极具分享欲的人格报告。
推演 (Sandbox)：用户在报告页点击“带着你的DNA开启推演”，无缝跳转至沙盘。系统自动继承用户的DNA属性作为游戏初始参数，让用户看到不同选择下的平行人生。
掌控 (Compass)：当用户在沙盘中体验到“财富自由”的爽感后，引导其下载/使用资产罗盘，开始记录真实账单，将虚拟的成就感转化为现实的执行力。

️ 技术架构关联 (Technical Architecture)
采用“独立仓库 + 核心算法抽离”的轻量级解耦架构，确保各端开发与迭代的灵活性：
独立部署：Oracle、Sandbox、Compass 分别拥有独立的 Git 仓库和前端项目（如 Vite/Next.js/React Native），互不干扰，可独立上线。
数据流转：模块间通过 URL 加密参数传递核心状态（如 DNA Code、基础财务参数），实现零后端成本的跨应用跳转。
引擎复用：底层计算逻辑（如复利计算、资产推演算法）将被抽离为独立的 TypeScript 核心库 (engine-core)，供 Sandbox 和 Compass 共享调用，保证数学模型的一致性。

** 美术视觉关联 (Visual Identity)**本板块相关规范与具体资产将由美术团队全权负责制定与产出。开发团队在初期搭建框架时，仅需关注布局结构与交互动效的逻辑占位。

UI/UX 交互参考标准：
    参考对象：多邻国 (Duolingo)。
    核心借鉴点：答题流程的极致流畅度、选项点击后的即时正向反馈、无压力的闯关体验以及清晰的 UI 层级。
    执行说明：开发同学在编写 Oracle 模块时，应优先复刻这种丝滑的交互手感（如选项切换的过渡动画、进度条的动态反馈等）。具体的色彩体系、图标风格、字体排版及插画资产，待美术同学输出设计规范后统一接入。
    注意移动端的适配，确保手机的答题和结果展示足够流畅
